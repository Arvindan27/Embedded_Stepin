\begin{DoxyNote}{Note}
For 0-\/200 ADC value duty cycle of PWM is 20\%, giving output as 20 degree Celsius 

For 210-\/500 ADC value duty cycle of PWM is 40\%, giving output as 25 degree Celsius 

For 510-\/700 ADC value duty cycle of PWM is 70\% , giving output as 29 degree Celsius 

For 710-\/1024 ADC value duty cycle of PWM is 95\% , giving output as 33 degree Celsius 
\end{DoxyNote}
